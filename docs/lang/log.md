log keyword
===========

The `log` keyword logs its parameters to STDOUT. It accepts multiple parameters, separated by a space or indention.

Syntax
------
```
log param1 [param2] [paramN]
log
  param1
  [param2]
  [paramN]
```

#### Firescript

```fire
log 'Foo' bar 'bla'
```

#### Javascript

```js
console.log('Foo', bar, 'bla')
```
