Objects
=======

```fire
const obj = { foo: 'Foo', bar: 'Bar' }
const obj = {
  foo: 'Foo',
  bar: 'Bar'
}

const obj =
  foo: 'Foo'
  bar: 'Bar'
```

Complex objects
---------------

```fire
const obj =
  foo:
    bar: 'Bar'
    bla:
      'bli'
      'bla'
      'blu'
      blobber:
        'blubber'
```
