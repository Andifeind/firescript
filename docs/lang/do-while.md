do...while
==========

A **do...while** statement loops through its `body` until a `test` expression evaluates into false

Syntax
------

```
do
  [body]
while [test]
```

#### FireScript

```fire
do
  i += 1
  console.log(i)
while i < 100
```

#### Javascript

```js
do {
  i += 1;
  console.log(i);
} while (i < 100)
```
